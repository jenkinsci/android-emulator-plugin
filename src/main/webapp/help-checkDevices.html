Select this option, if you may have physical devices connected to the server, and only want to start an emulator if they are not connected.

When this option is checked, the plugin will run the 

adb -d devices
command to see if there are any physical devices connected before starting an emulator. It will only start a new emulator if there are no devices connected. The adb server port number option, specifies which port the adb server is using.
