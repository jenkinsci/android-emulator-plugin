Selecting this option will examine the output of the Android Application Exerciser Monkey tool and publish a summary to the build page.
If it is determined that monkey crashed the application or a not-responding situation occurred, the build will be marked as unstable, unless another "Set build result" option is chosen.

Unless a filename is entered below, the output will be read from a file called "monkey.txt" in the root of the build workspace.

To run monkey automatically on an Android emulator or device and write the results to a file, select "Add build step" above and choose "Run Android monkey tester".
